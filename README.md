# Customer-behavior
Data Analytics project showcasing Customer Behavior Analysis using Python, SQL, and Power BI
This project presents a complete analytics workflow designed for a real-world retail company to uncover customer behavior patterns and shopping trends. It combines Python-based data analysis, SQL-driven business insights, and Power BI dashboarding to support executive decision-making.

Objectives  
- Analyze customer purchase behavior and shopping trends  
- Identify high-value customer segments and product preferences  
- Solve business problems using SQL queries and data modeling  
- Visualize key performance indicators (KPIs) for stakeholders  

Tools & Technologies  
- Python: Data cleaning, transformation, and exploratory data analysis (Pandas, NumPy, Matplotlib)  
- SQL: Business logic implementation, data extraction, and modeling  
- Power BI: Interactive dashboards for executive reporting  
- Jupyter Notebook / VS Code: Development environment  

Key Features  
- Cleaned and prepared raw retail datasets for analysis  
- Performed EDA to uncover trends in customer behavior and product sales  
- Solved business queries using SQL (e.g., customer lifetime value, repeat purchase rates)  
- Built relational data models to connect customer, transaction, and product data  
- Developed Power BI dashboards to visualize:  
  - Sales trends by category and region  
  - Customer segmentation and retention  
  - Purchase frequency and basket size  

Business Impact  
- Enabled marketing teams to target high-value customers more effectively  
- Supported sales strategy with data-driven insights  
- Improved executive visibility into customer trends and performance metrics
